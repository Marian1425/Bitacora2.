Propiedad position.

Ideas principales.

  • Especifica el de posicionamiento utilizado para un elemento.
  • Los elementos HTML se posicionan estáticos de forma predeterminada.
  • Un elemento con position: static; no está posicionado de ninguna manera especial.
  • Un elemento con position: relative; está posicionado en relación con su posición normal.
  • El desplazamiento de la caja se controla con las propiedades top, bottom, left, right y z-index.
  • El posicionamiento absoluto se emplea para establecer de forma exacta la posición en la que se muestra en la caja de un elemento.
  • Un elemento con positiom: fixed; siempre permanece en el mismo lugar, incluso si la página se desplaza.
  • Un elemento sticky alterna entre relativo y fijo, dependiendo de la posición de desplazamiento.

Apuntes de la clase.

Propiedad Position. Especifica el tipo me método de posicionamiento utilizado para un elemento.

La nueva posición del elemento se indica mediante las propiedades top, bottom, left, right y z-index. que en este caso depende del posicionamiento del elemento contenedor cuando tiene una posición diferente a static.
Cuando un elemento se posiciona de forma absoluta, el resto de elementos se posiciona de forma absoluta, el resto de elementos de la página se ven afectados y modifican su posición. Los demás elementos ocuparán el espacio que deja el elemento desplazado y es probable que se produzcan solapamientos con otros elementos.

Propiedad display.

Ideas principales.

  • Es una de las más complejas de CSS.
  • La propiedad display establece el tipo de la caja que genera cada elemento.
  • display: block; Siempre comienza en una nueva línea y ocupa todo el ancho disponible.
  • Un elemento inline no comienza en una nueva línea.
  • inline-block; Permite establecer un ancho y una altura en el elemento.
  • display: none; Se usa comúnmente en JavaScript para ocultar y mostrar elementos cuando el usuario realiza alguna acción como pulsar un botón o unos enlaces sin eliminarlos ni crearlos nuevamente.
  • display: table; Permite asignar un comportamiento de tabla a los elementos que hay dentro.

Apuntes de la clase.

Propiedad display. Es una de las más complejas de CSS. La propiedad display establece el tipo de la caja que genera cada elemento. Cada elemento de HTML tiene un valor de visualización predeterminada para la mayoría de los elementos es block o inline.

Propiedad display: block; Siempre comienza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y hacia la derecha tanto como sea posible).

Propiedad display: inline; Un elemento inline no comienza en una nueva línea y solo ocupa el ancho que sea necesario. Ej: span, a, img.

Propiedad display: inline-block; Permite establecer un ancho y una altura en el elemento. Además, se respetan los margin y padding, que con display: inline no son respetados. No agrega un salto de línea después del elemento, por lo que el elemento puede alinearse junto a otros elementos.

Propiedad display: none; Se usa comúnmente en JavaScript para ocultar y mostrar elementos cuando el usuario realiza alguna acción como pulsar un botón o un enlaces sin eliminarlos ni crearlos nuevamente. Hace que el elemento no genere ninguna caja. El resultado es que el elemento desaparece por completo de la pagina y no ocupa sitio, por lo que los elementos adyacentes ocupan su lugar. Se utiliza sobre un elemento todos sus descendientes también desaparecen por completo de la página.

Propiedad display: table; Permite asignar un comportamiento de tabla a los elementos que hay dentro (anidados a la etiqueta donde estas colocando el display: table). Dentro de este elemento de pueden usar otra series de valores de display que corresponden con los roles de distintas etiquetas HTML que usas en las tablas.

display: table-row; Actúa como un elemento fila, etiqueta ‘tr’

display: table-cell; Etiquetas que trabajan como una celda de tabla, etiqueta ‘td’.

Resumen.

La propiedad display establece el tipo de la caja que genera cada elemento. Cada elemento de HTML tiene un valor de visualización predeterminada para la mayoría de los elementos es block o inline. Existen varios elementos de display y cada uno cuenta con sus propias funciones y especificaciones, por ejemplo, display: block; display: inline; display: inline-block; display: none; display: table;.

Propiedades de CSS3.

Ideas principales.

  • CSS3 no solo cubre diseño y estilo, sino que también cubre forma y movimiento.
  • Se pueden generar esquinas redondeadas, sombras y transformaciones.
  • Se pueden cargar y usar cualquier fuente que necesitemos.
  • Se pueden agregar varios colores a un mismo elemento como un círculo o un cuadrado.
  • También se puede modificar la forma de un elemento ya que se puede rotar, inclinar, trasladar o escalar.
  • Además, se puede crear un borde a una imagen personalizada.
  • Se le puede generar un segundo borde alejado del borde original de algún elemento.
  • Se puede especificar la opacidad de un color.

Apuntes de la clase.

Propiedades de CSS3. En un intento por reducir el uso de código JavaScript y para estandarizar funciones populares, CSS3 no solo cubre diseño y estilo sino también forma y movimiento.
Desde esquinas redondeadas y sombras hasta transformaciones y reposiciones de los elementos ya presentados en pantalla, cada posible efecto aplicado previamente utilizando JavaScript fue cubierto.

Border-radius. Genera esquinas redondeadas para la caja formada por el elemento. Posee dos parámetros diferentes que dan forma a la esquina.

Box-shadow. Esta propiedad crea sombras para la caja formada por el elementó.

@font-face. Esta regla nos permite cargar y usar cualquier fuente que necesitemos.

linear-gradient. Esta función puede ser aplicada a las propiedades background o background-image para generar un gradiente lineal.

radial-gradient. Esta función puede ser aplicada a las propiedades background o background-image para generar un gradiente radial.

RGBA. rgba() esta función es una mejora de rgb() con un canal alfa que especifica la opacidad de un color.

HSLA. Significa tono, saturación y luminosidad.

Outline. Esta propiedad fue mejorada con la incorporación de otra propiedad llamada outline-offset. Ambas propiedades combinadas generan un segundo borde alejado del borde original del elemento.

Border-image. Esta propiedad crea un borde con una imagen personalizada.

Transform. Esta propiedad modifica la forma de un elemento.

Resumen.

Las nuevas propiedades del CSS3 se hicieron para reducir el uso de JavaScrip y para estandarizar funciones populares, CSS3 no solo cubre diseño y estilo sino también forma y estilo. Con estas nuevas funciones se pueden redondear las esquinas, poner sombras, transformar y hacer reposiciones de los elementos sin la necesidad de utilizar JavaScript. Estos cambios convierten a CSS3 en una tecnología prácticamente inédita.

Animación.

Ideas principales.

  • Con CSS se pueden animar la mayoría de los elementos sin usar JavaScript.
  • Para poder hacer que una animación funcione, se debe vincular la animación a un elemento.
  • Existen diferentes etiquetas para definir la animación, por ejemplo, para especificar el tiempo que debe durar la animación, si la animación debe reproducirse hacia adelante, hacia atrás o en ciclos alternos.
  • También de puede especificar el numero de veces que se debe reproducir la animación o especificar la curva de velocidad de la animación, entre otros.
  • Se pueden usar varias etiquetas para hacer una animación todo depende de lo que tú quieras realizar.

Apuntes de la clase.

Animación. CSS permite animar la mayoría de los elementos sin usar JavaScript.
animation-duration: especifica cuanto tiempo debe transcurrir una animación.
animation-direction: especifica si debe reproducirse hacia adelante, hacia atrás o en ciclos alternos.

  1. normal: la animación se reproduce de forma normal (hacia adelante).
  2. reverse: la animación se reproduce en dirección inversa (hacia atrás).
  3. alternate: la animación se reproduce hacia adelante primero, luego hacia atrás.
  4. alternate-reverse: la animación se reproduce hacia atrás primero, luego hacia adelante.
animation-iteration-count: especifica el número de veces que se debe reproducir.
animation-delay: Especifica un retraso para el inicio de una animación.
animation-play-state: especifica si la animación se está ejecutando o está en pausa.
animation-timing-fuction: especifica la curva de velocidad de la animación.
  1. linear: especifica una animación con la misma velocidad de principio a fin.
  2. ease: especifica un ciclo lento, luego rápido y luego termina lentamente.
  3. ease-in: especifica una animación con un inicio lento.
  4. ease-out: especifica una animación con un final lento.
  5. ease-in-out: especifica una animación con un inicio y un final lentos.
  6. cubic-bezier (n, n, n, n): permite definir sus valores en cubic-bezer.
animation-fill-mode: especifica un estilo cuando la animación no se está reproduciendo.
  1. none: valor predeterminado.
  2. forwards: retendrá los valores de estilo establecidos por el ultimo fotograma clave.
  3. backwards: obtendrá los valores establecidos por el primer fotograma clave.
  4. both: seguirá las reglas tanto para adelante como para atrás, extiende las propiedades de animación en ambas direcciones.
animation: propiedad rápida para configurara todas las propiedades de la animación.

Resumen.

Para animar en CSS ya no es necesario usar JavaScript. Si especificas un estilo CSS dentro de la regla @keyframes, la animación cambiara de estilo actual al estilo nuevo. Para que una animación pueda funcionar, se debe vincular la animación a un elemento. Con las diferentes etiquetas con las que cuenta CSS se le puede dar la forma que desees a la animación.

Referencias avanzadas de CSS3.

Ideas principales.

  • Existen diferentes dos tipos de atributos el id y el class, el primero sirve para darle un nombre que identifique al elemento, y el segundo es más flexible y puede ser asignado a varios elementos.
  • Hay varios tipos de Pseudoclases las cuales nos permiten encontrar uno o varios hijos, cada una cuenta con sus funciones y especificaciones.
  • También contamos con los selectores los cuales nos permiten dar estilo a los elementos que especifiquemos con este atributo.

Apuntes de la clase.

Atributo id. Es como un nombre que identifica al elemento.

Atributo class. Es más flexible y puede ser asignado a varios elementos HTML en el documento que comparten un diseño similar.

Elementos. Un selector de elementos en CSS es una regla que selecciona todos los elementos de un tipo especifico en el documento HTML.

Pseudoclases

:nth-child(). Nos permite encontrar un hijo especifico.
:nth-child(odd) :nth-child(even). Las palabras clave odd y even se utilizan para seleccionar elementos con base en su posición en la secuencia de hijos.
:first-child. La pseudoclase first-child referencia solo el primer hijo.
:last-child. La pseudoclase last-child referencia solo el ultimo hijo.
:only:child. La pseudoclase only-child afecta un elemento siempre y cuando sea el único hijo disponible.
:not(). Nos permite declarar una excepción.

Selector atributo.

Un selector de atributo en CSS selecciona elementos HTML en función de la presencia o el valor de uno de sus atributos.
[atributo^=“inicio”]. Una regla con el selector (^=) selecciona elementos cuyo atributo comienza con un valor especifico.
[atributo$=“final”]. Una regla con selector ($=) selecciona elementos cuyo atributo termina con un valor especifico.
[atributo*=“cadena”]. Una regla con ele selector (*=) selecciona elementos cuyo atributo contiene una cadena especifica en cualquier posición.

Selectores de relación.

Los selectores de relación en CSS permiten seleccionar elementos basados en su relación con otros elementos dentro del documento.
Hijos directos (>). Selecciona los elementos que son hijos inmediatos de un elemento padre.
hermanos adyacentes (+). Selecciona un elemento que es hermano inmediato, es decir, está justo después de otro elemento.
Hermanos generales (~). Selecciona todos los elementos que son hermanos de un elemento anterior, es decir, esta antes, sin importar sin son hermanos inmediatos o no.

Resumen.

CSS también cuenta con diferentes etiquetas, cuenta con dos atributos, el id y el class. Además, cuenta con diferentes Pseudoclases las cuales nos permiten encontrar a un hijo especifico, esto dependiendo de la pseudoclase que nosotros utilicemos. También hay diferentes selectores los cuales nos permiten aplicar estilos a los elementos que especifiquemos utilizando los diversos selectores con los que contamos.
